# Linux 环境部署脚本

## 环境

环境为 centos/debian

## 脚本

仅需使用以下命令即可，快速配置对应的环境，但是注意，目前尚未完成回滚机制。以下为已经完成的脚本：

- Docker

```shell
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```

- Nodejs

```shell
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```

- 前端

```shell
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```

- 后端

```shell
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```
